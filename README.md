## Usagem

Seu computador deve ter `php-cgi` instalado para o funcionamento correto do servidor web.

```bash
$ git clone https://github.com/mikaelmello/lumify
$ cd lumify
$ mkdir obj files
$ make
$ ./lumify
```

Abra o endereço `http://localhost:8080` no seu browser de preferência.
